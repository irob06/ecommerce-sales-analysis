# E-Commerce Sales Analysis

## Overview
This project analyzes e-commerce transaction data to find key business insights
using Python, Pandas, and Matplotlib.

## Dataset
- 15,000 rows of transaction data
- Columns: customer_id, product_id, basket_date, basket_count

## Tools Used
- Python 3.11
- Pandas
- Matplotlib

## Key Findings
- Total unique customers: 13,871
- Total unique products: 13,161
- May 2019 had the highest orders: 19,209
- June 2019 orders: 13,097
- Most active customer placed 29 orders
- Top product sold 69 units

## Charts
- Top 10 Products by Quantity
- Monthly Orders Trend
- Top 10 Active Customers

## How to Run
1. Clone the repository
2. Install dependencies: `pip install pandas matplotlib`
3. Run: `python analysis.py`
